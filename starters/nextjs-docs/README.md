# Next.js Docs Starter

This starter shows how RepoDocs AI assets can be integrated into a Next.js documentation site.

## Included Files

- `package.json` starter scripts
- `next.config.ts` baseline config
- `src/app/page.tsx` docs home page
- `src/app/docs/page.tsx` workflow summary page
- `src/app/globals.css` starter styling

## Intended Use

1. Copy the starter into a Next.js repository.
2. Move approved RepoDocs AI Markdown content into the site content pipeline of your choice.
3. Extend routes, content sourcing, and theme behavior for your product docs.
