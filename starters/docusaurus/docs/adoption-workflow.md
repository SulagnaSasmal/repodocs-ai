# Adoption Workflow

## Step 1

Copy the templates, prompts, validation assets, and examples you want to publish.

## Step 2

Create or generate Markdown content using RepoDocs AI workflows.

## Step 3

Place approved content into the Docusaurus `docs/` directory and update `sidebars.ts`.

## Step 4

Build and publish the site after validation passes.
