# Docusaurus Starter

This starter shows how RepoDocs AI assets can be published inside a Docusaurus documentation site.

## Included Files

- `package.json` with Docusaurus starter scripts
- `docusaurus.config.ts` with basic site configuration
- `sidebars.ts` for starter navigation
- `docs/intro.md` landing page
- `docs/adoption-workflow.md` integration guide
- `docs/api-docs-workflow.md` API documentation publishing flow

## Intended Use

1. Copy this starter into a repository that will host a Docusaurus docs site.
2. Bring over RepoDocs AI templates, prompts, validation assets, and examples that match your domain.
3. Adjust the site title, navigation, theme, and repository links.
