# GitBook Starter

This starter shows how RepoDocs AI assets can be published in a GitBook workspace.

## Included Files

- `SUMMARY.md` starter navigation
- `README.md` home page
- `getting-started.md` adoption page
- `api-docs-workflow.md` API authoring and publication workflow

## Intended Use

1. Copy these files into a GitBook-managed repository.
2. Bring over the RepoDocs AI templates, prompts, and validation assets you need.
3. Expand navigation and ownership metadata for the target product team.
