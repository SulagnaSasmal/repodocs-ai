# Getting Started

## Step 1

Copy the `templates`, `prompts`, and `validation` assets you want to use into your documentation repository.

## Step 2

Create product or API docs using the matching RepoDocs AI templates.

## Step 3

Add the resulting Markdown pages into the MkDocs `docs/` tree and update `mkdocs.yml` navigation.

## Step 4

Run your validation and review workflows before publishing.
