# RepoDocs AI MkDocs Starter

This starter demonstrates how a repository can publish RepoDocs AI assets through an MkDocs site.

## What To Bring In

- templates from the main RepoDocs AI repository
- examples that match your API or product domain
- review checklists for authoring and approval

## Why This Starter Exists

It gives teams a fast path from repository-native docs assets to a browsable documentation site.
