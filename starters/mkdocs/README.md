# MkDocs Starter

This starter shows how RepoDocs AI assets can be dropped into an MkDocs-based documentation site.

## Included Files

- `mkdocs.yml` with a minimal navigation structure
- `docs/index.md` as the landing page
- `docs/getting-started.md` for adoption guidance
- `docs/api-reference.md` showing how to link back to repository templates and examples

## Intended Use

1. Copy the contents of this directory into a repository that uses MkDocs.
2. Move or adapt RepoDocs AI templates and examples into the destination docs tree.
3. Adjust navigation, theme settings, and repository links for the target product.
