# Archive

This section keeps the earliest RepoDocs AI product-spec documents for historical reference.

If you are evaluating the current product, start with [`docs/product-specification.md`](../product-specification.md) instead.

## Archived Files

- `product-spec-overview-v1.md`: Version 1 product overview spec covering audience, problem, solution, and feature set
- `product-spec-system-v1.md`: Version 1 system spec covering architecture, template library, prompt framework, metadata schema, and pricing

## Why Keep These Files

These documents were part of the original design work. They remain available for historical context, but new readers should start with the current specification.

Their ideas informed the current implementation and the canonical specification.
