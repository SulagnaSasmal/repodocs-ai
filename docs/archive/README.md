# Archive — Original Product Specification Documents

This directory contains the original product specification documents that were
written during the initial design phase of RepoDocs AI.

These files are preserved for historical reference. The canonical, up-to-date
specification lives in [`docs/product-specification.md`](../product-specification.md).

## Files

| File | Description |
|------|-------------|
| `product-spec-overview-v1.md` | Version 1 product overview spec — audience, problem, solution, and feature set |
| `product-spec-system-v1.md` | Version 1 system spec — architecture, template library, prompt framework, metadata schema, and pricing |

## Why These Exist

Both documents were originally committed with `.litcoffee` file extensions and
non-standard filenames (including a `#` character in one name). They have been
moved here, renamed to valid Markdown filenames, and preserved so the original
product thinking is not lost.

The content in these files informed the current implementation and the
`docs/product-specification.md` canonical source.
