# Prompts

This directory contains AI prompt packs for drafting and reviewing documentation.

Use prompts together with the templates, not as a substitute for them.

Recommended flow:

1. Choose a template from `../templates/`.
2. Use the matching prompt to draft or review content.
3. Run `npm run validate` before publishing.

Prompt categories in this directory:

- `api-generation/` for OpenAPI-to-documentation workflows
- `product-docs/` for product overview, user guide, and administrator guide generation
- `feature-docs/` for component-to-feature and workflow documentation
- `review/` for documentation-review and quality checks
