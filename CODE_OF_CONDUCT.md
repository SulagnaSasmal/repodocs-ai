# Code of Conduct

RepoDocs AI is intended to be a respectful, practical project for contributors, reviewers, and users.

## Expected Behavior

Contributors are expected to:

- communicate clearly and respectfully
- focus criticism on the work, not the person
- assume good intent while still being technically rigorous
- keep discussions constructive and relevant to the project
- help maintain a welcoming environment for developers, writers, and maintainers

## Unacceptable Behavior

The following behavior is not acceptable in project spaces:

- harassment or personal attacks
- hateful, discriminatory, or abusive language
- deliberate intimidation or disruption
- sharing private or sensitive information without permission
- bad-faith behavior intended to derail collaboration

## Enforcement

Project maintainers may edit, remove, or reject contributions and discussions that violate this code of conduct.

## Reporting

If you experience or observe behavior that undermines the project environment, report it to the repository maintainer through GitHub issues or direct repository contact channels when appropriate.
