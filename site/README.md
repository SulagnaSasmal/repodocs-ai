# Site

This directory contains a static landing page for RepoDocs AI.

## Purpose

- present the product as a public-facing repository
- provide a lightweight GitHub Pages-ready marketing surface
- summarize roadmap progress and buyer positioning

## Deployment

The repository includes `.github/workflows/deploy-site.yml` to deploy this directory through GitHub Pages.

If GitHub Pages is enabled for the repository, pushes to `main` will publish the site automatically.
