# Site

This directory contains a static landing page for RepoDocs AI.

## Purpose

- present the product as a public-facing repository
- provide a lightweight GitHub Pages-ready marketing surface
- summarize roadmap progress, automation coverage, and buyer positioning

## Deployment

The repository includes `.github/workflows/deploy-site.yml` to deploy this directory through GitHub Pages.

If GitHub Pages is enabled for the repository, pushes to `main` will publish the site automatically.

## Pages

- `index.html`: landing page and product overview
- `ready-to-install-system.html`: long-form installable system walkthrough
- `product-specification.html`: published product specification page
- `roadmap.html`: roadmap and specification coverage summary
- `spec-summary.html`: strict roadmap/spec summary in published HTML form
- `automation.html`: automation workflow and validation overview
