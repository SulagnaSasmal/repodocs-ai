Control Plane

Repository automation now has a small hosted runtime surface.

RepoDocs AI can now expose validation, export, agent, analytics, and graph jobs through a small Node control plane with HTTP endpoints and a lightweight dashboard.

Endpoints

Runnable over HTTP

  • GET /health
  • GET /jobs
  • GET /runs
  • GET /artifacts
  • POST /jobs/validate
  • POST /jobs/export
  • POST /jobs/automation

Hosted Scope

Small by design

The control plane is meant to bridge repo-native automation and a hosted deployment model without introducing a full platform stack. It runs one job at a time, keeps artifacts on disk, and exposes the results through JSON.

Runtime

Start it with one command.

Run npm run control-plane:start to launch the dashboard and API on port 4312 by default.
