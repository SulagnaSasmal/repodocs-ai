# Product Overview

## Purpose

RepoDocs AI helps SaaS teams create structured technical documentation in repositories using reusable templates and AI prompts.

## Target Users

- engineering teams
- developer relations teams
- technical writers
- product owners supporting API programs

## Business Problem

Teams often maintain inconsistent documentation across repositories, which slows onboarding, weakens API usability, and creates review overhead.

## Core Capabilities

- reusable Markdown documentation templates
- AI generation prompts with review guardrails
- metadata-driven documentation structure
- example docs for faster adoption

## Product Positioning

RepoDocs AI is positioned as a practical docs-as-code system for SaaS and API teams that want more structure than a folder of templates but less overhead than a full documentation platform.

## Dependencies

- GitHub repositories
- Markdown-based documentation workflows
- static site tooling such as MkDocs or Docusaurus
