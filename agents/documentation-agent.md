# Documentation Agent Blueprint

## Purpose

This asset defines a future AI documentation agent mode for RepoDocs AI.

## Responsibilities

- classify documentation requests
- choose the correct template and prompt pack
- collect source material
- generate a draft
- run validation and review prompts

## Inputs

- OpenAPI specification or product notes
- target audience
- repository context

## Outputs

- Markdown draft
- validation findings
- missing SME input list
