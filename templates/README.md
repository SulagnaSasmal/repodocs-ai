# Templates

RepoDocs AI ships reusable documentation templates grouped by job, not by generic file type.

Start with:

- `api/` for API overview and endpoint documentation
- `features/` for feature and workflow documentation
- `governance/` for review and quality guardrails

Also available:

- `product/` for product, user-guide, and admin documentation
- `operations/` for runbooks and deployment guidance
- `architecture/` for system and integration design docs
